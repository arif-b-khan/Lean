# Lean Engineering Documentation

This documentation site is configured with MkDocs.

## Local Preview

1. Install dependencies:

```bash
pip install mkdocs
```

Optional enhanced theme:

```bash
pip install mkdocs-material
```

2. Start local docs server:

```bash
mkdocs serve -a 127.0.0.1:8000
```

3. Open in your browser:

- http://127.0.0.1:8000

## Generated Feature Documents

Agent-generated feature documentation is stored under:

- `Docs/generated/<feature-name>/index.md`

Assets should be stored under:

- `Docs/generated/<feature-name>/assets/`
